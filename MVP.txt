  Взятие Квестов за очки ВВС
  Отчетность по выполнению
    
    * Страница получает от сервера массив. 
    * В массиве квесты у которых просрочены отчеты. 
    * Для каждого квеста строится блок с табличкой выполнений 
        и с кнопками напротив каждого выполнения.
    * Кнопки активны только в первых рядах табличек. 
    * Востальных активируются последовательно, 
        после отчета по предыдущему выполнению
    * По каждому нажатию, запрос на сервер
    * Счетчик очков сверху
    * Отменяемости пока не будет
    


  Получение очков ВВС при положительном выполнении

  /Взятие меньшего кол-ва квестов, меньшей сложности на меньший срок

    * В.п. прогресса общего прогресса
